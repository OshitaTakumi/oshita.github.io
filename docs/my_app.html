学部2年次後期

開発アプリ：お絵かきソフト
開発言語

: java
講義にて初めてjavaに触れ，アプリ制作を経験．講義内でいくつかの練習問題をこなした後に最終課題としてオリジナルアプリを作成． 最終課題の開発期間は約二か月．
開発の経緯

周りでは最終課題にシューティングゲームを作成している学生が多かった．他の学生と開発途中で互いに困っている点や，改善案について話し合う中で ゲーム内で用いる画像でいいものが見つからないという声が多数あった．そのため，自分でicon等を作成できるアプリを作成しようと考え，アイコンを 簡単に自作できるようなお絵かきソフトの開発を決めた．

学部3年次前期

開発アプリ：クリップボード共有アプリ
開発言語

: java : java script
講義にてAndroid Studioを用いてjavaを用いてAndroidアプリを作成．最終課題としてオリジナルアプリを作成．
javaでスマートフォン側のコードを開発，java scriptを用いてGoogle chromeの拡張機能を作成．

開発の経緯

大学のシステムにアクセスする際に多要素認証が導入され，２段階のパスワード認証が必要となった．認証パスワードには MicrosoftのAuthenticatorが用いられ，このアプリは１端末にしか入れることができない．
多くの学生はスマートフォンにこのアプリを入れていたが大学のシステムにアクセスする際はPCを用いることが多く，パスワードの確認 にいちいちスマートフォンを開く必要があった．これに対して，スマホとPC(windows)間でクリップボード共有を行えるアプリの作成を決めた．
